To add/edit policy:
-   macOS: MonoDevelop-Unity -> Custom Policies...
- Windows: Tools -> Custom Policies...

To set default policy:
-   macOS: MonoDevelop-Unity -> Preferences...
- Windows: Tools -> Options...
- In this new window: Source Code -> Code Formatting
- Set both C# & Text to use KRG policy, and Name Conventions too, then hit OK.

To apply policy to current solution:
1. Project -> Apply Policy...
2. Project -> Solution Options

macOS Policy location: ~/Library/MonoDevelop-Unity-5.0/Policies
